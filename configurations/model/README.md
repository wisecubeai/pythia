# pythia-llamafile
Run your own OpenAI API locally. Tested to work on Mac OS X & Linux. 

***build***:
  - make sure you have docker installed
  - build the image
```
./build.sh
```

***run***:
```
./run.sh
```
***UI Test***:  
  - go to http://127.0.0.1:8080
  - say 'hello'! 

***API Test***:  
```
python -m client
```
